# Email Priority Classification System

## ğŸ“Œ Overview
The Email Priority Classification System is a machine learning application that analyzes email text and classifies messages into **Urgent**, **Normal**, or **Low Priority** categories. The system uses semantic text embeddings generated by a pre-trained MiniLM model and a Logistic Regression classifier trained on labeled email samples.

A Streamlit-based dashboard allows users to upload datasets, view predictions instantly, and download the results as a CSV file.

---

## âœ… Features
- Upload custom CSV files containing email subject and body
- Select a preloaded sample dataset
- Automatic text preprocessing and cleaning
- Semantic embeddings using `all-MiniLM-L6-v2`
- Urgency prediction using Logistic Regression
- Results displayed in tabular format
- Bar chart visualization of class distribution
- Downloadable output CSV with predicted labels

### Go to VS Code terminal and clone the repository

git clone https://github.com/riyaacodes/email-priority-classification-system.git

### Enter into the project root folder
cd email-priority-classification-system

### Install dependencies
pip install -r requirements.txt

### Train the model (run once)

### Launch the Streamlit app
streamlit run app.py
